feems.SpatialGraph¶

class feems.SpatialGraph(genotypes, sample_pos, node_pos, edges, scale_snps=False)[source]¶
__init__(genotypes, sample_pos, node_pos, edges, scale_snps=False)[source]¶

Represents the spatial network which the data is defined on and stores relevant matricies / performs linear albegra routines needed for the model and optimization. Inherits from the networkx Graph object.

Parameters

  • genotypes (numpy.ndarray) – genotypes for samples

  • sample_pos (numpy.ndarray) – spatial positions for samples

  • node_pos (numpy.ndarray) – spatial positions of nodes

  • edges (numpy.ndarray) – edge array

  • scale_snps (Bool) – boolean to scale SNPs by SNP specific Binomial variance estimates
